How to use the web page Play?

  • Figure below represents the start page when the web play page loaded.

  • Enter integer positive value between 1 to ∞ such as: 1,2,3,...,∞. The positive number you enter represents the size of rubik's cube.

  • After that press enter button, it will show you orthographic's graph for rubik's cube depending on the dimension you enter. In addition, another two new buttons will show on the page.

  • If you press rotate or scramble button, a new reset button will show on the page.

  • To start the game, press scramble button. It will show you orthographic scramble graph.

  • Below button, we have four variables to choose to start the game. Let first press reset button.

  • After we choose these variable or not it has a default value. We can press rotate button.

  • For choose axis rotation. There are X, Y, and Z.

  • For choose direction of rotation. There are clock wise and anti clock wise.

  • For choose step of rotation. There are 1, 2, and 3.

  • For choose position of cell to be rotate. There are 1,2,...,n where n represents size of rubik's cube that enter in first step.

  • Press reset button will initialize the color of faces to the default color before start the game. Also, these color faces represent the target of gamer that need to change the cell location of rubik's cube.
