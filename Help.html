How to use the web page Play?

  1. Enter interger positive value between 1 to ∞ such as: 1,2,3,...,∞. The positive number you enter represents the size of rubik's cube.

  2. After press enter button, it will show you orthographic's graph for rubik's cube depending on the deminsion you enter. Inaddition, another two new button will show on the page.

  3. If you press rotate or scramble button, a new reset button will show on the page.

  4. To start the game; press scramble button. It will show you orthographic scramble graph.

  5. Below orthographic graph; we have four variable to choose to start the game. After we choose these variable or not it has a default value. We can press rotate button.

  6. For choose axis rotation. There are X, Y, and Z.

  7. For choos direction of rotation. There are clock wise and anti clock wise.

  8. For choose step of rotation. There are 1, 2, and 3.

  9. For choose position of cell to be rotate. There are 1,2,...,n where n represents size of rubik's cube that enter in first step.

  10. Press reset button will intialize the color of faces to the default color before start the game.
